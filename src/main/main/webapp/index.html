Hotel Booking API

Welcome to the Hotel Booking API. This API provides endpoints for managing users, payments, and reviews.

API Endpoints

User Endpoints

  • GET /api/users/{id} - Get user by ID
  • POST /api/users/login - User login
  • POST /api/users/register - User registration

Payment Endpoints

  • GET /api/payments - Get all payments
  • GET /api/payments/{id} - Get payment by ID
  • GET /api/payments/refundable - Get refundable payments
  • GET /api/payments/status/{status} - Get payments by status
  • POST /api/payments - Create a new payment
  • POST /api/payments/{id}/refund - Process a refund

Review Endpoints

  • GET /api/reviews - Get all reviews
  • GET /api/reviews/approved - Get approved reviews
  • GET /api/reviews/{id} - Get review by ID
  • GET /api/reviews/user/{userId} - Get user reviews
  • POST /api/reviews - Create a new review
  • PATCH /api/reviews/{id}/approve - Approve a review
